# egui-app

Cross platform egui application template,

- eframe
- egui
- egui-phosphor
- egui-tiles
- egui-logger
- egui-file-dialog
- egui-plot

### Desktop Application

Windows and Linux

```sh
# Development: Builds to 'target/debug/'
cargo run

# Distribution: Builds to 'target/release/'
cargo build --release
```
